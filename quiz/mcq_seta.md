# Software Agile Project Management Principle	
## Multiple-Choice Questions: Set A - Software Agile Project Management Principle**

1. **Which of the following is a key principle of Agile project management?**  
   A) Detailed upfront planning  
   B) Continuous delivery of valuable software  
   C) Rigid documentation requirements  
   D) Waterfall methodology  

2. **What is the primary focus of Agile project management?**  
   A) Delivering comprehensive documentation  
   B) Customer collaboration and satisfaction  
   C) Avoiding change during the project  
   D) Meeting predetermined budgets  

3. **In Agile, project identification involves:**  
   A) Creating a fixed timeline  
   B) Identifying project goals and stakeholders  
   C) Analyzing every possible risk upfront  
   D) Documenting every aspect in detail  

4. **Which document is most critical in Agile planning?**  
   A) The comprehensive project plan  
   B) A light, flexible roadmap  
   C) Detailed requirement documents  
   D) Fixed scope agreements  

5. **Agile monitoring typically uses which of the following?**  
   A) Static charts  
   B) Iterative feedback loops  
   C) End-of-project reviews  
   D) Gantt charts only  

6. **What is the primary goal of documenting in Agile?**  
   A) To meet legal requirements  
   B) To ensure flexibility and clarity  
   C) To create a blueprint for the entire project  
   D) To reduce communication with the team  

7. **Which Agile principle encourages adapting to change?**  
   A) Fixed planning  
   B) Iterative progress with feedback  
   C) Avoidance of change  
   D) Linear progression  

8. **During project analysis in Agile, teams primarily focus on:**  
   A) Risk elimination  
   B) Ongoing value delivery  
   C) Creating extensive documents  
   D) Determining fixed budgets  

9. **Agile promotes project treatment by:**  
   A) Adhering to strict controls  
   B) Emphasizing flexibility and adaptability  
   C) Avoiding stakeholder feedback  
   D) Locking down project scope  

10. **Which of these is NOT a characteristic of Agile documentation?**  
    A) Lightweight and minimal  
    B) Easily adaptable  
    C) Rigid and exhaustive  
    D) Focused on value addition  

11. **Agile project monitoring emphasizes:**  
    A) Real-time tracking and iteration  
    B) Fixed timelines and milestones  
    C) Comprehensive end-of-cycle reviews  
    D) A single, fixed progress report  

12. **What concept is at the core of Agile project management?**  
    A) Delivering perfect software on the first attempt  
    B) Iteration and feedback-driven development  
    C) Avoiding interaction with stakeholders  
    D) Ensuring rigid conformity to plans  

13. **How are risks managed in Agile projects?**  
    A) Identified upfront and fixed for the duration of the project  
    B) Iteratively identified, assessed, and addressed  
    C) Avoided entirely by rigid planning  
    D) Delegated solely to upper management  

14. **What role do stakeholders play in Agile project management?**  
    A) Minimal interaction after initial planning  
    B) Continuous collaboration and feedback  
    C) Approval at the project‚Äôs conclusion  
    D) Defining rigid project documentation  

15. **Why is flexibility important in Agile project management?**  
    A) To increase complexity in projects  
    B) To adapt to changing requirements and priorities  
    C) To minimize stakeholder involvement  
    D) To ensure project budgets are always met  

16. **Agile project documentation should focus on:**  
    A) Providing legal evidence  
    B) Supporting collaboration and iteration  
    C) Preventing change during development  
    D) Listing all possible risks upfront  

17. **Which Agile concept promotes early and continuous delivery?**  
    A) Timeboxing  
    B) Incremental value delivery  
    C) End-of-project analysis  
    D) Comprehensive pre-planning  

18. **Which phase involves identifying key project stakeholders?**  
    A) Monitoring  
    B) Planning  
    C) Identification  
    D) Treatment  

19. **What type of planning does Agile use?**  
    A) Fixed, detailed upfront planning  
    B) Incremental and adaptive planning  
    C) Comprehensive and static planning  
    D) Planning only at the start and end  

20. **The Agile Manifesto values:**  
    A) Processes over individuals  
    B) Contracts over collaboration  
    C) Comprehensive documentation over working software  
    D) Responding to change over following a plan
